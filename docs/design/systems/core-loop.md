# 核心循环简介

一局游戏开始时，玩家将选择携带一些装备和物资进入**随机生成**的“地牢”探索；

玩家将在“地牢”中随机遭遇敌人、发生事件、遇到宝物等等；

“地牢”由多个“层级”（level）组成，随着层级的深入，难度会越来越高

“地牢”是一个“图”（大多数情况下是规则的方格图），有房间（chamber）、通道（passage）、出入口（entrance & exit）等；

“地牢”的房间中可能出现敌人、事件、宝物；

玩家与敌人战斗的形式是回合制卡牌对局；

玩家可以使用多种装备，玩家的装备槽位依据身体部位进行划分，例如左右手可以装备武器、盾牌等，头部、胸部、腿部、足部等可以装备防具，以及可以装备饰品等；

玩家的装备可能会给玩家的卡组中加入手牌，例如玩家装备一把“直剑”可能给玩家提供若干张“斩击”，玩家装备一个“木盾”可以给玩家提供若干张“防御”。反之，脱下装备会从玩家的卡组中移除对应添加的卡牌；

玩家的装备可能会给玩家提供属性值加成，例如“斗篷”可能会给玩家提供+1敏捷。

玩家的装备可能会给玩家提供某种“能力（Ability）”，例如“回复项链”可能给玩家提供“在战斗开始时，你恢复3点生命值”这样的“能力”；

玩家的角色具有多个属性维度，例如力量、敏捷、智力等；属性值的来源有多种，例如玩家角色本身固有的数值、装备提供的属性加成、消耗品提供的加成、特殊事件提供的加成（虽然用词是“加成”，但本意应该是"modify"，效果可正可负）；

玩家角色的属性有多种效用：

- 例如根据装备的属性补正修改装备提供的卡牌的效用，例如一把武器”流星锤“提供若干张”打击“牌，”打击“的基础面板是5点伤害，属性补正是0.5倍力量，玩家的力量是12，假设力量的中性值是10，那么”打击“的实际面板是5 + 0.5*（12-10）= 6；
- 影响随机事件的发生与效果等；

玩家的战斗循环如此：

1. 开始时玩家从卡组中抽取若干张初始手牌；
2. 玩家每回合开始有一定的”费用“，用于支付打出手牌的消耗；
3. 玩家的回合开始时会从抽牌堆抽取若干张牌加入手牌；
4. 在玩家的回合中，玩家可以消耗费用以打出手牌，来对敌人造成伤害、进行防御、施展特殊效果；
5. 敌人在其回合会尝试攻击玩家或做出特殊行动；
6. 玩家的回合结束时，将手牌中的所有牌放入弃牌堆；
7. 当玩家尝试抽牌时，如果玩家的抽牌堆为空，则触发将弃牌堆洗牌后放到抽牌堆；
8. 当玩家或全部敌人一方死亡，或发生特殊事件，一局战斗结束；

玩家战胜敌人将获得战利品，包括新的装备、资源、货币等等；

玩家在“地牢”的特定节点或特殊事件中可以选择**携带战利品**离开地牢，玩家可以在地牢外变卖战利品，或者将其保留，留作下一局游戏的初始装备。

玩家局内获得战利品时需要将其装入背包才能带走，而玩家的背包大小有限

玩家获取的战利品有多种，例如：

- 货币，可用于局内或局外购买物品
- 装备
- 消耗品，提供一次性的效果
- 素材？用于制作物品
- 换金道具？用于兑换成货币
