# 装备与卡牌系统

#### 装备

玩家和部分单位拥有装备槽位，分为左右手、头部、胸部、腿部、足部，以及若干饰品槽位。



配备上”装备“时会为玩家卡组添加卡牌，反之，脱除装备时将移除对应的卡牌。

玩家在战斗时无法选择穿脱装备，而在战斗外的探索环节中的非休息时间中，穿脱装备需要消耗一点行动力，这一点行动力给出的结果是在玩家下一次消耗任何行动力之前，玩家可以随意穿脱装备，也就是说这一点行动力不是为穿脱某一件装备所消耗的成本，而是开启了一个“换装备”的动作序列，只是这个动作序列中玩家也大可以选择只换一件装备。而在休息时间中，玩家可以没有成本地随意更换装备。



“装备”拥有耐久度，一旦耐久度降至0，就会永久损坏，强制移除出玩家的装备槽位并永久销毁。装备的耐久度损耗的基本条件是经历战斗，会根据战斗的烈度减少装备的耐久度。



装备分为普通装备和地牢装备，普通装备既可能在局外获取，也可能在地牢中获得，并且可以带出地牢；而地牢装备只会在地牢中出现，并且无法带出地牢，相应地，由于地牢装备获取难度更高，其强度也应该更强。

相对来说，普通装备的强度表现得相对均衡，拥有较强的正面效果的同时可能会有较为温和的负面效果，但地牢装备可能会在提供更为激进的正面效果的同时附加更为糟糕的负面效果。



左右手可装备武器、盾牌等装备。

”武器“类装备主要为玩家提供攻击性的卡牌，次要提供其他类型卡牌，例如一件武器可能是这样的：

“制式长剑”：提供3张“斩击”，1张“准备架势”。

​	“斩击”：1行动力，造成5点物理伤害。

​	“准备架势”：1行动力，你的下一次攻击额外造成4点物理伤害。

左右手装备槽位的激活情况和装备有关，同时也会影响装备提供的卡牌，对于一些武器，必须双手使用，此时装备到左手和右手均等价于双持，此时另一只手槽位被禁用，不允许装备更多手持装备。

对于一部分单手装备，如果当前左右手只装配了这一件装备，那么自动变为双持单不会禁用另一只手槽位，单持或双持单手武器提供的卡牌有所不同，例如双持“制式长剑”时，为了保证玩家的防御能力，会额外提供2张1费防4的防御。并非所有单手装备都可以双持，例如对于“匕首”之类的装备，双持似乎并不是一个合理的设计。

双持一些装备可能会改变装备的属性补正等级。

双持有时是一种”选择“而有时是一种”必要“，这意味着对于一些装备，玩家可以选择要双持或单持，例如上面制式长剑的例子，玩家可以在缺少盾牌时选择双持从而获得防御能力，而对于一些装备，玩家只能单持或双持，例如在上面的匕首的例子中，玩家只能单持匕首，而考虑一些重武器，例如长柄斧，这种重武器只能双持显然是合理的设计。



“盾牌”类装备主要为玩家提供防御性卡牌，次要提供其他类型卡牌，例如一件盾牌可能是这样的：
“厚重盾牌”：提供3张“防御”，1张“盾击”。

​	“防御”：1行动力，获得4点格挡值。

​	“盾击”：1行动力，造成2点伤害，使目标陷入持续1回合的“失衡”，失衡状态下的目标受到的伤害增加25%，造成的伤害减少25%。



头部、胸部、腿部、足部等部位可统一归类为身体部位类装备，可为玩家提供各种类型卡牌。

其中头部、胸部更多提供防御性卡牌和功能性卡牌，以及属性值加成。

腿部、足部可以提供各种类型卡牌，以及属性值加成。

例如可能有这样的一套装备：

头部：“军官头盔”：+1魅力，+1体质，+1护甲，提供1张”振奋演说“。

​	”振奋演说“：1行动力，本回合友方获得临时的1点力量加成，抽2张牌。

胸部：“僧侣斗篷”：+1智慧，+1感知，-1魅力，提供1张“祷告”

​	“祷告”：1行动力，使1位友方获得4点格挡值，并且抽1张牌。

腿部：“猎人皮裤”：+1敏捷，提供1张“追击”。

​	“追击”：0行动力，本回合中你每打出一张牌，对随机敌人造成2点伤害。

足部：“流氓布鞋”：+1力量，-1魅力，提供1张“蛮横飞踢”。

​	“蛮横飞踢”：2行动力，造成8点伤害，使目标陷入持续2回合的“失衡”。



饰品类装备可以为玩家提供各种卡牌，以及一些特殊规则效果例如：

“血蛭项链”：+1体质，效果：你每击败一个敌人，你回复3点生命值。

“贵族胸针”：+1魅力，效果：在商人处给出此道具，可以让该商人售出的物品价格减少50%。

值得注意的是，在以上的描述中为了减少重复没有对所有类型装备描述其提供的所有卡牌类型和效果，实际上，所有装备拥有相同的本质，均可以产生各种类型的卡牌，以及提供特殊效果，只不过侧重点不同，例如考虑一件胸部装备：

”秘银刺甲“：+1敏捷，+1护甲，提供1张”尖刺撞击“。效果：敌人对你造成伤害时，其同时受到3点物理伤害。

”尖刺撞击“：1行动力，获得4点护甲，下一次你受到敌人的伤害时，它同时受到3点伤害。

（以上仅仅只是一些举例，不代表具体设计）



除了提供属性值加成和卡牌之外，”装备”还有几项重要的要素：

1. 装备提供的卡牌可能拥有“属性补正”，“属性补正”会参与卡牌效果的数值计算，分为多个等级，可能是这样的：

   设：面板属性值 - 属性中性值 = d 

   卡牌效果基础数值计算函数统一为：

   f(卡牌面板数值，总属性值，属性补正) = （卡牌面板数值 * 1）+ g(d, 属性补正)

   其中g是属性补正函数，当d为正数，也即总属性值是正向值，取属性补正对应的正向部分作为计算因子，乘上|d|

   即g(d, 属性补正) = |d| * 属性补正正向取值， d >= 0；|d| * 属性补正负向取值, d < 0

   值得注意的是，g的两种情况均严格向下取整，即令增益更不显著，减益更糟糕。

   以下属性补正把正向因子记作+XX%，负向记作-XX%

   1. 无补正
   2. A，+100%，-100%
   3. B，+75%，-50%
   4. C，+50%，-50%
   5. D，+25%，-0%

   补充说明：如上显示的属性补正因子，可见在总属性值低于中性值时，A级补正惩罚最为严重，每低1点属性值，最终卡牌基础数值需-1；而对B，C级，只有当每低1或2点属性值时，卡牌数值才会-1；D级则没有惩罚。

   属性补正不一定总是对单个属性计算，也可能多个属性合在一起使用一个属性补正等级，例如B力量与敏捷补正，则总属性值=力量+敏捷，而非单独计算

2. 不同装备可能提供同名的卡牌但卡牌拥有不同的面板数值和属性补正等级，例如“制式长剑”提供3张1费打4的“斩击”，而“侍卫钢刀”可能提供3张1费打（4+A力量补正）的“斩击”

3. 同名装备可能提供不同卡牌，包括卡牌名不同、面板不同、属性补正等级不同，这些因素由装备的稀有度决定。在一些稀有度下装备提供的卡牌面板、补正可能是固定的，例如“普通XXX”可能只会出固定的卡牌，而在另外一些稀有度下装备可能提供不固定的数值。

4. 双手以及部分身体部位的装备槽位未装配时也会提供类似穿戴力量了装备的效果，可以视作穿戴了一项隐形的装备，这是为了保证玩家总是拥有一定的战斗能力以及不至于太小的卡组，例如空手时，其背后的装备可能就是“空手”，双手“武器”（一旦穿上手持装备就会替换），提供若干张“打击”和“防御”。



#### 卡牌

卡牌的基本要素：

花费（Cost）和卡牌效果。

Cost的基本要素是“行动力”（见战斗系统），可能有其他类型的Cost。



卡牌效果有非常多种可能性，给出一些基本例子：

1. 造成伤害
   1. 对单个目标造成单次伤害
   2. 对单个目标造成多段伤害
   3. 对单个目标造成随机伤害
   4. 对多个目标造成上述两种情况的伤害
   5. 对多个目标随机造成伤害（和造成随机伤害不同）
2. 获得格挡
   1. 使单个目标获得一次格挡
   2. 使单个目标获得若干次格挡
   3. 使单个目标获得随机格挡
   4. 使多个目标获得以上情况的格挡
   5. 使多个目标随机获得格挡
3. 抽牌
4. 弃牌
5. 移动牌的位置
   1. 将弃牌堆的牌随意置入抽牌堆
   2. 将弃牌堆中的牌放到抽牌堆特定位置
   3. 将抽牌堆中特定位置的牌放到抽牌堆的特定位置
6. 使出牌者失去生命值
7. 施加正面/负面效果（buff/debuff）
8. 消耗/获得Cost
9. 以上效果的组合
