# 属性系统

游戏中的单位有各种“属性”，其分为不同的层级，基础属性（Primary Attribute），派生属性（Secondary Attribute）以及其他不好分类的属性。



#### 基础属性

基础属性可分为以下几个

物理属性：

- 力量
- 体质
- 敏捷

精神属性：

- 智慧
- 感知
- 魅力

基础属性可以参与派生属性的计算（见下）、装备和卡牌效果的计算（见卡牌与装备系统）、一些事件的判定。



基础属性有“中性值”的概念，可以认为其描述的是基于游戏世界观中的某个标准定义的一种生物属性的”平均值“，例如假设我们认为中性值表达的是”成年人类平均的属性值“，那么例如说中性值均取10，那么就可以认为正常成年人类的力量量化值是”10“，那么对于一只老鼠，其力量可能就是2，3之类很低的值；对于一个巨人族，力量可能就是20这种很高的值。取中性值为10只是为了观感上让高于中性值和低于中性值非常容易辨认，因为中性值取10，所以一位数就很显然低于中性值，两位数而非10的显然高于中性值。

属性值最低为0，这意味着该单位没有任何这一项属性的能力。例如一个细菌的智慧可以被认为是0，它只按照生物本能行动，可以被认为没有智慧。

此外中性值还与“属性补正”的计算有关（见装备系统章节），例如属性值的中性值是8，那么高于8的部分会参与属性补正的加成运算，低于8的部分会参与减益运算，以下在讨论属性补正时，如无特别说明，正向部分记作+P，负向部分记作-N。



#### 派生属性

- 生命值，由“体质“决定，通过某种计算公式可从”体质“计算出生命值的基础数值，此公式应当满足边际效益递减
- 速度，由物理属性和角色穿戴的装备的重量共同决定，和战斗开始的先手权有关



#### 其他属性

- 行动力
- 伤害(Damage/DamageToDeal)
- 将要吸收的伤害(DamageToTake)
- 格挡
- 将要获取的格挡(BlockToGain)
- 幸运
- 护甲
- XX属性倍率
- XX属性修正
- XX属性偏移
- 伤害吸收率
- 伤害吸收偏移



- XX属性倍率：单位某种属性等的一种乘法因子，一般内部按加法计算，例如两个效果"伤害倍率+25%"，“伤害倍率+50%”，假设初始伤害倍率是100%，最后算出来应该是100% + 25% + 50% = 175%
- XX属性修正：单位某种属性的一种乘法因子，一般内部按乘法计算，例如两个效果“伤害修正1.25倍”“伤害修正1.15倍”，假设初始修正是1.0倍，最终结果是1.0 * 1.25 * 1.15 = 1.4375
- XX属性偏移：单位某种属性的一种加法因子，一般内部按加法计算，例如两个效果“造成伤害+1”造成伤害+2“，那么造成的伤害的最终计算结果需要+3

这几个属性参与XX属性的更新计算，例如假设力量属性的基础值是12，现有属性倍率125%，属性修正1.3，属性偏移+1，那么力量属性的当前值就是12 * 125% * 1.3 + 1

“幸运”是一个比较特殊的属性，它可以影响各种随机事件发生的可能性。
