# 目标游戏体验

1. 丰富的随机性

   地牢关卡随机，敌人随机，装备随机（卡组随机），事件随机，每一局都可以是不同的体验

2. 丰富的策略选择

   开局装备选择、每一层地牢的路线规划、装备build、战斗局内卡牌打法、在有限背包大小的限制下战利品的抉择、撤离时机的选择（是继续冒险深入还是见好就收？）都是潜在的策略选项
