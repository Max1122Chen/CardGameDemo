# 战斗系统

#### 战斗循环

战斗以卡牌对战的形式，以“回合”作为时间单位。

玩家的战斗循环如此：

1. 开始时玩家从卡组中抽取若干张初始手牌；
2. 玩家每回合开始总是会获得一定的”费用“，用于支付打出手牌的消耗；
3. 玩家的回合开始时会从抽牌堆抽取若干张牌加入手牌；
4. 在玩家的回合中，玩家可以消耗费用以打出手牌或者使用背包中的道具，来对敌人造成伤害、进行防御、施展特殊效果；
5. 敌人在其回合会尝试攻击玩家或做出特殊行动；
6. 玩家的回合结束时，将手牌中的所有牌放入弃牌堆；
7. 当玩家尝试抽牌时，如果玩家的抽牌堆为空，则触发将弃牌堆洗牌后放到抽牌堆；
8. 当玩家或全部敌人一方死亡或达到特殊条件，或发生特殊事件，一局战斗结束；



进入战斗时，会根据玩家和敌人的速度或战斗前的关系决出先手权。



#### NPC

游戏中的敌人、中立、友好单位均统称为”NPC“，这是广义的NPC的含义，狭义上来讲NPC仅指那些能够与玩家”平和“交互的单位。

NPC和玩家一样都有属性系统中的各项属性。

##### AI

本质上讲，其他单位和玩家在战斗中可以做出的行动集合是相同的，它们背后也拥有一套卡组，并且在战斗时也遵守着抽牌、花费行动力行动的逻辑。

单位的AI可以分成几种：

- 固定型，单位总是会采取设计好的固定策略的行动而不会有“智能”，这种设计一般运用在设定上智力低下的单位，例如史莱姆，并且它们的卡组往往非常小且固定，使得每回合
- 智能型，单位会真正结合此时战斗状况和手牌情况来做出决策，一般运用在设定上具有智能的单位，例如人类，并且它们的卡组可能会有一定规模
