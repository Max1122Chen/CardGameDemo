# 物品系统

游戏中的货币、装备、道具、高价值物品等都是“物品”，类别有具体其子类。

道具的子类：

- 消耗品
  - 食物
  - 药水
  - 其他
- 非消耗品

物品均有其“体积”和”质量“，表现为在背包和仓库中，其需要占据容器中有限的“格子”。

物品分为可堆叠和不可堆叠，如“金币”这种物品是可堆叠的，而像装备这种物品均是不可堆叠的。

物品分为不同的稀有度，分为：

- 普通
- 罕见
- 稀有
- 传奇

以便直观地说明物品的强度和掉落概率。
